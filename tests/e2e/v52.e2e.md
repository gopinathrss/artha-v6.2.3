# V5.2 — E2E (Playwright)

## Scope

Minimal **slice** that runs in CI or locally with `.env` and a reachable DB:

- `GET /healthz` — process + DB liveness.
- `GET /settings` — HTML shell loads; **App preferences** copy visible.
- `GET /api/app-settings` — JSON contract for dashboard boot.

## Run

```bash
npm run test:e2e
```

Uses `playwright.e2e.config.ts` (starts `tsx src/api/server.ts` unless `PW_REUSE_SERVER=1`).

## Extend

Add flows for Integrations **Save/Test** only against a disposable DB (mutates `IntegrationProvider`). Prefer `vitest` + `supertest` for pure API contracts; reserve Playwright for HTML + critical paths.
